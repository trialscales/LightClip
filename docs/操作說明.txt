
LightClip v1.4 操作說明
=======================

最後更新時間：2025-11-29 05:38:19

一、基本使用
------------
1. 啟動程式後，LightClip 會常駐於系統托盤，並自動監聽剪貼簿。
2. 主視窗分為兩個分頁：
   - 剪貼簿：顯示歷史記錄。
   - 模板：管理常用文字片段與回覆格式。
3. 上方搜尋列可輸入關鍵字，搭配類型篩選（文字 / 圖片 / 網址 / 檔案 / 釘選）來快速找到項目。
4. 介面字體為 18px 放大版，適合長時間工作使用。

二、剪貼簿分頁
--------------
- 自動記錄：當你在系統中複製文字、圖片、檔案或網址時，LightClip 會自動新增一筆記錄。
- 項目內容：左側為列表，右側顯示預覽與內容文字；若為圖片會顯示縮圖預覽。
- 釘選：對重要項目按「釘選 / 取消釘選」，即可避免被自動清除。
- 複製：按下「複製到剪貼簿」，可將選取項目重設為目前系統剪貼簿內容。
- 清除歷史：按下「清除歷史 (保留釘選)」，會移除所有非釘選的記錄。
- 文字自動換行：預覽區使用 18px 字體並自動換行，方便閱讀長文。

三、模板分頁
------------
- 新增模板：可為常用開場白、簽名、客服回覆等建立模板。
- 編輯模板：隨時修改名稱與內容。
- 刪除模板：移除不再使用的模板。
- 快速貼上：按「貼上模板」會將該模板內容寫入剪貼簿，之後在任何程式中 Ctrl+V 即可貼上。
- 設定快捷鍵：可將模板綁定 1~9 的索引，配合 Ctrl+Shift+數字 使用。

四、全域快捷鍵
--------------
- Ctrl+Shift+C ：顯示 / 還原 LightClip 主視窗。
- Ctrl+Shift+1~9 ：若對應索引綁定模板，則將該模板內容複製到剪貼簿。

注意：部分系統可能需要以系統管理員身分執行，keyboard 套件才能正常註冊全域快捷鍵。

五、設定
--------
可從上方選單「設定」開啟設定視窗，內容包含：
- 介面語言：繁體中文 / English。
- UI 主題：支援 6 組主題（深色、閱讀、科技藍綠、淺色、奶油、清爽模式）。
- 圖示主題：淺色圖示 / 深色圖示。
- 最大歷史筆數：限制未釘選歷史數量，避免佔用過多資源。
- 設定表單的上下間距約為 7px，畫面較為舒適易讀。
