LightClip 操作說明（v1.6）
================================

一、基本概念
-----------
LightClip 是一款「輕量型剪貼簿工具」，會在背景自動記錄你複製過的內容，
包含文字、圖片、連結等，方便你在之後快速找回並再次貼上。

主要特色：
- 自動保存多筆剪貼簿（預設 100 筆，可於「設定」修改）
- 支援釘選常用項目，避免被新資料覆蓋
- 可搜尋歷史記錄
- 模板功能可存放簽名、固定回覆格式等
- JSON 儲存，適合備份與版本控管
- 支援多主題與多語系
- 提供雲端同步匯出，方便搭配雲端硬碟使用

三、剪貼簿功能
-------------
1. 自動紀錄
   - 當你在系統中按下「複製」（Ctrl+C / 右鍵複製）時，LightClip 會自動記錄內容。
   - 支援類型：文字、圖片、連結。
   - 圖片會儲存到 data/images 下，並建立對應路徑。

2. 搜尋
   - 在「剪貼簿」上方輸入關鍵字，即可即時篩選列表。
   - 搜尋會比對預覽文字與完整內容。

3. 列表操作
   - 點選某一項，底部預覽區會顯示完整內容。
   - 按「複製選取內容」，可將該項再次複製到系統剪貼簿。
   - 按「刪除」可移除該筆歷史。
   - 按「釘選 / 取消釘選」可將項目固定在清單前端，不受歷史數量裁切影響。
   - 雙擊某一項也可以快速複製。

四、模板功能
-----------
1. 模板列表
   - 顯示所有已建立的模板。
   - 點選後下方會顯示內容預覽（字型為 13px）。
   - 雙擊模板即可開啟「模板編輯器」。

2. 新增 / 編輯 / 刪除
   - 「新增模板」：建立一個新的模板。
   - 「編輯模板」：編輯目前選取的模板。
   - 「刪除模板」：移除選取模板。

3. 模板複製
   - 在模板列表中按下「複製模板內容」，會將該模板的內容複製到剪貼簿，方便貼上。
   - 在模板編輯器中，名稱列右側也有「複製」按鈕，可隨時複製當前編輯中的內容。

4. 模板編輯器介面
   - 第一行：名稱（左）＋ 複製按鈕（右）。
   - 第二段：內容輸入區，可自動換行。
   - 第三段：下方檢視區，固定約 3 行高度、字體大小約 13px，方便確認排版。

五、設定
-------
- 最大剪貼簿記錄數：控制自動歷史的上限，僅會刪除未釘選的舊項目。
- 語言：在「繁體中文」與「English」之間切換。
- 顏色主題：在多種深色 / 淺色模式中切換，Menu Bar 也會跟隨主題變色。
- 快捷鍵提示：目前預設 Ctrl+Shift+C 可喚出主視窗（僅應用程式內），
  未實作全系統層級快捷鍵。

六、雲端同步
-----------
- 按下「雲端同步」會在 cloud/ 目錄下產生：
  - history_export.json
  - templates_export.json
  - settings_export.json
- 你可以將整個 LightClip 資料夾放進 Google Drive / Dropbox 等雲端同步資料夾，
  即可自動上傳備份。

1. 點選上方「問題回報」。 
2. 確認瀏覽器已開啟 Gmail 寫信畫面。
