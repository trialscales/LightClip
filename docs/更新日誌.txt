LightClip 更新日誌
==========================

v1.6.1
------
- 上方選單列改版：統一到標準 Menu Bar 中顯示「操作說明 / 更新日誌 / 問題回報 / 雲端同步 / 設定 / 關於 / 說明」。
- Menu Bar 會依照顏色主題自動切換淺色 / 深色風格。
- 模板列表支援「點兩下」開啟模板編輯器。
- 模板編輯器調整：名稱與「複製」按鈕同列顯示，下方加入 3 行 13px 檢視區。
- 剪貼簿列表與預覽區優化顯示，自動換行更舒適。
- 更新說明文件與關於視窗，列出所有主要功能。
- 更新日誌以「由新到舊」方式排列。

v1.5.0
------
- 新增頂部工具列按鈕：操作說明、更新日誌、問題回報、雲端同步。
- 問題回報可自動開啟 Gmail 寫信頁面並帶入收件人。
- 新增雲端同步功能：匯出剪貼簿、模板與設定為 JSON。
- 調整介面整體字體為 18px，更適合長時間閱讀。
- 新增模板編輯器視窗：可編輯內容並預覽。
- 新增模板一鍵複製功能。

v1.4.0
------
- 加入多種深色與淺色主題。
- 支援切換繁中 / 英文介面語言。
- 新增系統托盤圖示與背景常駐。

v1.3.0
------
- 提升剪貼簿歷史穩定性，避免重複條目。
- 新增搜尋功能，可依文字關鍵字篩選。

v1.2.0
------
- 新增模板管理分頁，支援新增 / 編輯 / 刪除模板。
- 模板適合用來儲存簽名檔與常用回覆。

v1.1.0
------
- 支援圖片與連結類型剪貼簿。
- 新增釘選功能，避免常用項目被清除。

v1.0.0
------
- LightClip 初版發布。
- 基本剪貼簿歷史記錄與複製功能。
