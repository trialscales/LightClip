
LightClip 更新日誌
==================

最後更新時間：2025-11-29 05:38:19

v1.0.0
------
- 初版剪貼簿工具。
- 自動監聽剪貼簿，支援文字 / 圖片 / 網址 / 檔案。
- JSON 本機儲存資料。
- 釘選功能與歷史清除。
- 系統托盤支援。

v1.1.0
------
- 新增多語系（繁體中文 / English）支援。
- 新增 icon_theme 設定，支援淺色 / 深色圖示主題切換。
- 新增多份 TXT 說明文件。
- 調整儲存結構，加入 language / icon_theme 設定項目。

v1.2.0
------
- 新增設定頁（語言、UI 主題、圖示主題、最大歷史筆數可調整）。
- 新增「關於」視窗，顯示版本資訊。
- 新增模板管理分頁，可建立 / 編輯 / 刪除常用文字模板。
- 模板支援 1~9 快捷鍵索引，配合全域快捷鍵 Ctrl+Shift+數字 使用。
- 新增 keyboard 套件，提供全域快捷鍵支援（Ctrl+Shift+C 叫出視窗）。
- 介面文字全面改用多語系字串。

v1.3.0
------
- 新增多主題系統，共 6 種 UI 主題配色（深色、閱讀、科技藍綠、淺色、奶油、清爽冷色）。
- UI 改為預設 18px 大字版，更適合長時間閱讀。
- 更新設定頁，可選擇具名主題（dark_default / light_default 等）。
- 新增「說明」選單與「問題回報」，一鍵開啟郵件寄至 trialscales0430@gmail.com。
- 統整 docs，顯示三大文件：操作說明、更新日誌、問題回報。
- 剪貼簿記錄改用 timestamp_local 與 timestamp_iso 欄位，保留更精確的時間資訊。

v1.4.0
------
- 更新 About 視窗，列出完整 LightClip v1.4 功能清單。
- 主視窗頂部新增三個按鈕：「操作說明」、「更新日誌」、「問題回報」。
- 操作說明與更新日誌按鈕會直接讀取 docs 底下對應 TXT 檔，並在新視窗中顯示內容。
- 問題回報按鈕改為開啟瀏覽器，使用 mailto 方式寄 Email 至 trialscales0430@gmail.com。
- 全介面維持 18px 放大字體，適合長時間使用。
- 設定視窗的控制項上下間距調整為約 7px，版面更舒適。
- 剪貼簿預覽區文字自動換行，閱讀長文更方便。
