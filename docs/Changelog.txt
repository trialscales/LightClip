
LightClip 更新日誌
==================

v1.0.0
------
- 初版剪貼簿工具。
- 自動監聽剪貼簿，支援文字 / 圖片 / 網址 / 檔案。
- JSON 本機儲存資料。
- 釘選功能與歷史清除。
- 系統托盤支援。

v1.1.0
------
- 新增多語系（繁體中文 / English）支援。
- 新增 icon_theme 設定，支援淺色 / 深色圖示主題切換。
- 新增 docs 資料夾，提供多份 TXT 說明文件。
- 新增 Changelog.txt 更新記錄。
- 調整儲存結構，加入 language / icon_theme 設定項目。

v1.2.0
------
- 新增設定頁（語言、UI 主題、圖示主題、最大歷史筆數可調整）。
- 新增「關於」視窗，顯示版本資訊。
- 新增模板管理頁籤，可建立 / 編輯 / 刪除常用文字模板。
- 模板支援 1~9 快捷鍵索引，配合全域快捷鍵 Ctrl+Shift+數字使用。
- 新增 keyboard 套件，提供全域快捷鍵支援（Ctrl+Shift+C 叫出視窗）。
- 介面文字全面改用多語系字串，利於後續擴充更多語言。
